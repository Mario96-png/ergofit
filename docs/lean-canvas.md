# Lean Canvas: AgentStore — Tiendas nicho automatizadas con IA

## Problem (Problema)
1. Montar una tienda online desde cero requiere tiempo y conocimientos técnicos (tienda, productos, logística).
2. El marketing en redes sociales es tedioso y requiere constancia diaria (crear contenido, publicar, interactuar).
3. Encontrar productos ganadores para dropshipping es difícil sin herramientas costosas o experiencia.

## Customer Segments (Segmentos)
- **Target primario:** Emprendedores individuales (tú) que quieren crear tiendas nicho sin invertir mucho tiempo.
- **Perfil:** Persona con curiosidad por el ecommerce, presupuesto ajustado, dispuesta a probar nichos.
- **Early adopters:** Tú mismo. Una vez que funcione para ti, podrías abrirlo a otros.

## Unique Value Proposition (Propuesta Única)
**"Tu agente de IA crea la tienda, encuentra los productos y gestiona el marketing. Tú solo eliges el nicho."**

Diferencia clave vs alternativas:
- Oberlo/Spocket: solo catálogo de productos, no crean la tienda ni hacen marketing.
- Shopify: plataforma, pero todo el trabajo es manual.
- Agencias de marketing: caras (500-2000€/mes).
- **AgentStore: automatización completa con IA desde el producto hasta el post en redes.**

## Solution (Solución)
1. **Agente investigador** → Busca productos ganadores para el nicho elegido (AliExpress, CJ, etc.)
2. **Agente constructor** → Monta la tienda online (Shopify / WooCommerce) con los productos seleccionados.
3. **Agente marketer** → Genera contenido orgánico (posts, stories, reels) y lo publica en IG + TikTok.
4. **Tú** → Supervisas, eliges nicho, y revisas resultados.

## Channels (Canales)
- Las propias redes sociales de cada tienda (IG, TikTok) → tráfico orgánico
- Posible SEO a medio plazo
- Boca a boca / nicho (ej: foros de fitness, grupos de belleza en Facebook)

## Revenue Streams (Ingresos)
- **Margen de dropshipping:** Diferencia entre precio de proveedor y precio de venta (~30-50% por producto).
- **Escala:** Múltiples tiendas → múltiples fuentes de ingresos.
- **Fase 2 (opcional):** Vender el sistema como servicio a otros emprendedores.

## Cost Structure (Costes)
| Concepto | Coste estimado |
|----------|---------------|
| Shopify (por tienda) | 29€/mes |
| Dominio + hosting | 15€/año aprox |
| Proveedor IA (API) | ~20-50€/mes (según uso) |
| Proveedor dropshipping | Coste del producto (variable) |
| Herramientas (Canva, etc.) | 0€ si usas el agente para diseño |
| **Total por tienda** | ~35-50€/mes |

## Key Metrics (Métricas Clave)
- **Nº de tiendas activas**
- **Pedidos/día por tienda**
- **Coste de adquisición de cliente (CAC) → CERO (orgánico)**
- **Margen bruto por producto**
- **Tiempo desde idea → tienda operativa** (objetivo: < 1 semana)

## Unfair Advantage (Ventaja Injusta)
- **Por ahora: ninguna.** Esto es temporal. Es un mercado muy competitivo.
- **A construir:** Datos propios sobre qué nichos y productos funcionan mejor. Cuantas más tiendas tengas, más datos acumulas y mejor afinas el sistema.
- **A largo plazo:** La red de agentes + el know-how de qué contenido funciona en cada nicho.
